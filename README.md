![Design preview for the Blog preview card coding challenge](./preview.jpg)

## 🚀 Overview

This project is a solution to the Blog Preview Card challenge. The goal was to recreate the design as closely as possible while maintaining clean, scalable code.

---

## 🛠️ Built With

* HTML5
* CSS3
* Flexbox
* CSS Variables

---

## 📚 What I Learned

* Structuring semantic HTML for better readability
* Using Flexbox for alignment and layout
* Managing spacing using `gap` instead of excessive margins
* Applying CSS variables for consistent design
* Building responsive layouts with a mobile-first approach

---

## 🎯 Features

* Responsive design
* Clean and minimal UI
* Hover interaction on the card
* Scalable styling using `rem` units
